# 안녕하세요, 상상을 현실로 바꾸는 개발자 이승헌입니다.

이 블로그는 **AI 기반 지식 그래프**를 활용하여 포스트 간의 연결을 시각화합니다.

## 주요 기능

1. **AI 임베딩**: OpenAI의 임베딩 모델을 사용하여 각 포스트의 의미를 벡터로 변환합니다.
2. **그래프 시각화**: D3.js를 활용한 인터랙티브 그래프로 포스트 간 연결을 탐색할 수 있습니다.
3. **다크 테마**: Claude에서 영감받은 모던한 다크 테마를 적용했습니다.

## 코드 예시

```typescript
const embedding = await openai.embeddings.create({
  model: "text-embedding-3-small",
  input: postContent,
});
```
